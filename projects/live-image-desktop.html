live web image as desktop

My main project at the moment

This is my favorite project at the moment. It is a simple windows program which downloads images from a given URL and sets them as your desktop background. It automatically detects when the image changes, thus making it perfect for things like live-webcams, or rapidly changing images. I wrote this in python. Soon I will compile it so that even those without python installed can benefit from it. If you do have python (and are willing to install pywin32), you can clone the project on Github and start right away.  It has a GUI, allowing you to edit the settings, and another program which runs in the background and changes your desktop background. If you have any suggestions for it, please contact me. For more technical stuff see the repo.
